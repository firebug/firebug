== Enablement UI ==
    * Disable always: when the Firebug UI is not active on any page, the debugger is disabled (minimal overhead)
    * Instant on: when the Firebug UI is active, HTML, CSS, DOM views activate (minimal overhead)
    * Script panel user-activation: initially disabled, enable, or initially enabled.
    * Net panel user-activiation: initially disabled, enable, or initially enabled.
    * No Allowed-sites/Disable for Site
