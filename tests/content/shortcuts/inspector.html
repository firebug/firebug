Test Case for Inspector shortcuts

This test-case is intended for Inspector shortcuts: Ctrl+Shift+C.
Jan Odvarko, odvarko@gmail.com

  1. Launch Firefox and make sure Firebug is deactivated for the current page.
  2. Press Ctrl+Shift+C.
  3. Firebug UI should be opened+activated and the Inspector activated (in inspecting mode).
  4. Press Ctrl+Shift+C again.
  5. Firebug UI should be still opened+activated and the Inspector deactivated.
