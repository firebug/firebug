Firebug Releases

The latest release is available on the front page.

Firebug 1.0.5

Firebug 1.0.5 (aka 1.05) was released in May 2007. This version has been widely used. Before reporting bugs, please try Firebug 1.1

Firebug 1.1 Betas

Recommended. Firebug 1.1.0 has had many beta tests and is the current stable version

Firebug 1.2 Alphas

Firebug 1.2 focuses on performance improvements. Currently unstable, usable, and bug reports are welcome.

ChromeBug Alphas

ChromeBug is Firebug for Firefox extensions. Currently unstable, barely usable, and comments welcome. The multi-XPI package for chromebug includes Firebug and ChromeList
