Issue 5507: Allow displaying original color values

This test also covers issue 6730 and issue 6406.

CSS panel

Steps to reproduce

  1. Open Firebug
  2. Switch to the CSS panel
  3. Select issue5507.html from the location menu
  4. Check the panel option Colors As Authored
    ⇒ All colors should be displayed as they were defined, i.e. the rules should look like this:
    #element1 { color: #f00; background: linear-gradient(135deg, rgb(180, 200, 255), #788cff) repeat scroll 0 0 blue; } #element1 { color: green; } #element1 { color: rgba(0, 0, 255, 1); }
  5. Check the panel option Colors As Hex
    ⇒ All colors should be displayed in hexadecimal format.
  6. Check the panel option Colors As RGB
    ⇒ All colors should be displayed in RGB format.
  7. Check the panel option Colors As HSL
    ⇒ All colors should be displayed in HSL format.

Expected result

  • The color values should be displayed according to the chosen setting.
