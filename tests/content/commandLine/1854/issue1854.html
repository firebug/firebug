Issue 1854: debug() sets breakpoints at comments

Steps to reproduce

  1. Open Firebug
  2. Enable the Script and Console panel
  3. Switch to the Console panel
  4. Reload the page
  5. Type debug(showOutput) into the Command Line and hit Enter
  6. Click the Show output button

Expected result

  • A breakpoint is set at line 11 (var output = document.getElementById("output");).
  • The script execution is halted at the breakpoint.
