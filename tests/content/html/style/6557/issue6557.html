Issue 6557: Font previews don't work

I am a remote font

I am a local font (using data-url)

Steps to reproduce

  1. Open Firebug, and switch to the HTML panel
  2. Inspect the first paragraph, whose text is: "I am a remote font"
  3. In the style panel, hover "Konstytucyja" (the value of the font-family property)
    ↳ The preview should display a text with the "Konstytucyja" font
  4. Inspect the second paragraph, whose text is: "I am a local font (using data-url)"
  5. In the style panel, hover "DeLarge" (the value of the font-family property)
    ↳ The preview should display a text with the "DeLarge" font
